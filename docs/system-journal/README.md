# System Journal Notes

Esta pasta guarda notas manuais por dia para complementar o journal automático.

- Cada ficheiro deve chamar-se `YYYY-MM-DD.md`.
- O gerador automático em [`/Users/joel/Documents/CoddingStuff/TorusAiTrading/scripts/update-system-journal.js`](/Users/joel/Documents/CoddingStuff/TorusAiTrading/scripts/update-system-journal.js) inclui estas notas na entrada diária correspondente.
- Usar esta pasta para registar contexto humano que não aparece bem em números, por exemplo:
  - avarias reais
  - decisões operacionais
  - exceções conhecidas
  - razões para uma mitigação manual
